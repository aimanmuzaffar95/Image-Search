# Image-Search
An image search app based on Unsplash's REST API using on Kotlin and Jetpack libraries

Tools and Libraries used in this project:
- Retrofit
- Coroutines
- MVVM Architecture
- Hilt Dagger
- Data binding
- Paging Library 2
